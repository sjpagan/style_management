INTRODUCTION
------------

This module use two external libraries to compile all files LESS and SCSS to CSS.

REQUIRMENTS
------------

require :
  1) leafo/scssphp:0.0.12
  2) oyejorge/less.php": "1.7.0.9

Manage download of this module by composer, it It will provide to download of required package and place it to vendor
folder.

INSTALLATION
------------

In drupal root folder Run command:
 - compores require drupal/style_management --with-dependencies

CONFIGURATION
------------

In configuration panel, have tree tabs:
 * SETTINGS
   - General Less Configuration
   - General Scss configuration
 * FILE MANAGER
   - List of compilable file
